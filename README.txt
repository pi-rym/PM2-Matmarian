Habilidades y competencias que desarrollarás...
Diseño de páginas web con HTML.

1.Clonar el repositorio del proyecto y explorar su estructura. Identificar los dos archivos package.json de ambas partes del proyecto.

2.Instalar las dependencias de la aplicación de Front-End y probar el funcionamiento con el comando “npm start”.
 Esto debería “levantar” nuestra aplicación de front en el navegador.

3.Diseñar la página web con la que estaremos trabajando. Se trata de una página de películas, por lo que el diseño
 debe ser acorde a la temática. Esto implicará:
a.Generar los archivos css necesarios y vincularlos al documento index.html.
b.Generar la plantilla HTML con sus elementos y asignar las clases y ids correspondientes para seleccionarlos y darles estilos.

4.Mapear los objetos incluídos en el array tempData para construir con sus datos las tarjetas correspondientes a cada película.
Incluir estas tarjetas en el contenedor que hayas creado.

5.Incluir en la página web enlaces a páginas estáticas. Estas páginas estáticas (mínimo 2) tendrán una temática a tu elección. 
Sugerencias: “Sobre mi proyecto”, “Historia del cine”, “Webs recomendadas”.

6.Configurar correctamente los enlaces desde index.html para poder navegar hacia estas páginas, y luego desde las mismas
 poder regresar al index.
