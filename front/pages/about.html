Mi proyecto consiste en una página web dedicada al mundo del cine, específicamente enfocada en películas. Utilicé tecnologías como HTML, CSS y JavaScript para desarrollarla. En esta plataforma, los usuarios pueden explorar una amplia variedad de películas, consultar información detallada sobre cada una, como título, año de lanzamiento y una breve descripción. Además, la página cuenta con un diseño atractivo y fácil de navegar, permitiendo una experiencia fluida para los usuarios. Con este proyecto, mi objetivo es brindar una plataforma intuitiva y entretenida para los amantes del cine, donde puedan descubrir nuevas películas y disfrutar de contenido de calidad.
